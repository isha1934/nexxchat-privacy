Datenschutzerklärung – Nexxchat

Letzte Aktualisierung: 2025

1. Verantwortliche Stelle

Nexxchat
Entwickler: Iskander Haddar
E-Mail: iskandar.haddar@gmail.com

2. Welche Daten werden erfasst?

2.1 Konto- und Login-Daten

  • E-Mail-Adresse
  • Passwort (verschlüsselt durch Firebase Authentication)
  • Nutzer-ID (UID)

2.2 Chat- und Kommunikationsdaten

  • Nachrichten (Text)
  • Bilder & Dateien (Firebase Storage)
  • Gruppeninformationen
  • Reaktionen, Timestamps, Verläufe
  • KI-generierte Bilder
  • Tetris-Scores / Mini-App-Daten
  • Video-Call Metadaten (Start, Ende, Dauer)

Wichtig: Keine Video- oder Audioinhalte werden gespeichert.

2.3 Geräte- & Nutzungsdaten

  • Firebase Installation ID
  • App-Version, Betriebssystem
  • Crash-Berichte (Crashlytics)
  • Analytics-Daten (optional)

3. Wofür werden die Daten verwendet?

  • Nutzerkonten-Verwaltung
  • Chats, Gruppen, Medien
  • Videoanrufe (ZegoCloud)
  • Funktionsverbesserung
  • Fehleranalyse
  • Synchronisation zwischen Geräten

4. Weitergabe der Daten an Dritte

Firebase (Google LLC)

  • Authentication
  • Firestore Database
  • Cloud Storage
  • Crashlytics / Analytics

Firebase Datenschutz: https://firebase.google.com/support/privacy

ZegoCloud

Für Videoanrufe wird ZegoCloud verwendet (Peer-to-Peer). Keine Video- oder Audioinhalte werden gespeichert.

Datenschutz: https://www.zegocloud.com/privacy

5. Speicherung der Daten

Alle Daten werden auf Firebase-Servern gespeichert (EU-Region, sofern konfiguriert).
Bilder & Dateien liegen in Firebase Cloud Storage.

6. Rechte gemäß DSGVO

  • Auskunft
  • Löschung des Kontos + aller Daten
  • Datenübertragbarkeit
  • Widerspruch
  • Beschwerderecht

7. Datenlöschung

Beim Löschen des Kontos werden alle personenbezogenen Daten, Chats, Bilder, Gruppendaten und Call-Logs vollständig aus Firebase gelöscht.

8. Sicherheit

Alle Verbindungen sind verschlüsselt (HTTPS). Firebase Security Rules schützen die gespeicherten Daten.
